Welcome to Java ReStart with WebFX!

The site presents two open source projects: WebFX and Java ReStart.

To run the samples below you will need Java 8 and a small executable jar that you may run as:

java -jar JavaRestart.jar

It will run WebFX browser (located remotely) from which you can run the samples below. (The samples will not run in a traditional browser.)

What is WebFX and Java ReStart

WebFX is an attempt to create a new web where HTML is replaced with FXML - the markup language for JavaFX UI, and the logic can be written in any programming language that is available on top of Java platform from JavaScript, Groovy, JRuby to Java, Kotlin and Scala. Java ReStart is a complimentary project that acts as transport layer for WebFX allowing FXML pages to reference arbitrary Java byte code on a web server and hence any custom JavaFX UI components and Java third-party libraries. In addition, Java ReStart allows to run arbitrary Java applications instantly from a web server downloading only required parts of applications on demand and executing them in parallel with downloading.

Some more details

Any scripting language that is available on of Java platform such as JavaScript, Groovy, JRuby can be used for writing UI logic in WebFX technology - the languages can be embedded transparently into FXML. This way you may employ rapid style development that developers used to use in the traditional web. For instance, Hello World sample below uses simple JavaScript code, while Hello Groovy sample uses simple Groovy script code.

Moreover, your FXML files may reference arbitrary Java byte code on a web server and hence any custom JavaFX UI components and Java third-party libraries via Java ReStart technology. So this way, you may write your client-side logic on any language that is available on top of Java platform such as Scala, Kotlin, whatever. For instance, REST demo can play music from remote server and the communication with the server is written in pure Java, while UI is handled by JavaScript. You may also check 2048 game that is pure Java code. In addition, Java ReStart allows to run arbitrary Java applications instantly from a web server downloading only required parts of applications on demand and executing them in parallel with downloading. This way applications run instantly like in the Web. For instance, you may run 2048 game not in a browser tab but as separate application.
